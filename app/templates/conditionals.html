Condicionales

En programación, una sentencia condicional es una instrucción o grupo de instrucciones que se pueden ejecutar o no en función del valor de una condición. Los principales tipos de condicionales son if, if else y switch.


if

La sentencia condicional if se usa para tomar decisiones, este evaluá básicamente una operación lógica, que de como resultado "verdadero" o "falso", y ejecuta la pieza de código siguiente siempre y cuando el resultado sea verdadero.

if ( 8 > 2 ) {
   miVariable = 'casa'
}


En ejemplo de arriba vemos que tras evaluar la operación 8 > 2 y comprobar que es verdadera, se ejecutará el trozo de código que están entre las llaves {...}. Si por el contrario, la operación hubera sido falsa, el fragmento de codigo que hay entre las dos llaves, no se habría ejecutado.


if else

La sentencia condicional if else es una evolución de la anterior. Podemos controlar cuan la operación de condición devuelve falso y ejecutar otro fragmento de código. De este modo tendremos controladas las dos situaciones, tanto si la condición es verdadera, como si la condición es falsa.

if ( miVariable > 2 ) {
   nombre = 'Juan'
} else {
   nombre = 'Pepe'
}


En el ejemplo anterior pasamos a evaluar si el dato que tenemos almacenado en la variable miVariable es mayor que 2. De ser así, se ejecutará el código que hay entre las dos primeras llaves, de lo contrario, pasará a ejcutarse el código que hay detrás la palabra reservada else. Los comparadores que podemos usar para evaluar las condiciones son:

  • > (mayor que)
  • < (menor que
  • == (igual que)
  • != (distinto que)
